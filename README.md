# Toko - React Login (Simple)

Ini adalah aplikasi demo halaman login sederhana menggunakan React (Vite).

Cara menjalankan:

1. Install dependensi:

```powershell
npm install
```

2. Jalankan development server:

```powershell
npm run dev
```

Lalu buka URL yang ditampilkan (biasanya http://localhost:5173).

Catatan: Ini adalah demo front-end saja; Anda perlu menambahkan backend untuk autentikasi nyata.
